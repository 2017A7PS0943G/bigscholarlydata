This directory contains implementation of the extension of the barabasi model to topics. 

driver.py : To run the topic prediction model

predict_topic_cites.py : Contains the functions to store and run the model

config.py : Contains database configurations and other settings for the model

model.py :  Contains the model definitions and helper functions

corpus.py : Contains the sqls and database access function

validation.py : Contains metric definitions for validation of model


To run the model, run driver.py and set the configurations in config.py
